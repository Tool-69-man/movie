# movie
swing小学期项目，电影购票jdbc
用户
1.javaSwing实现gif动画（简单进程切换图片）
2.轮播图，将图片地址存入数组，lable 监听器切换
3.用户直接订票，用户确定电影名称，时间，票价后，输入座位x,y。写下个人信息后，提交订单

管理员
电影，场次，订单，员工  CRUD,可以按数据库各字段查询
